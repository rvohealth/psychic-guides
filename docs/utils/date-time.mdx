# DateTime

Datetime fields in the database are returned as Luxon DateTime objects. DateTime provides robust handling of datetimes, including timezone manipulation, formatting, and math. See [https://moment.github.io/luxon/#/](https://moment.github.io/luxon/#/) for full documentation.

```ts
import { DateTime } from 'luxon'

DateTime.now()
DateTime.fromISO('2024-05-05T17:53:07.397Z')
```
