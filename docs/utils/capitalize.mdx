# capitalize

`capitalize` converts the first character of a string to uppercase.

```ts
import { capitalize } from '@rvoh/dream'

capitalize('helloWorld') // 'HelloWorld'
capitalize('dreamAndPsychic') // 'DreamAndPsychic'
```
