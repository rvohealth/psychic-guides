# compact

Returns a copy of an object or array, with `null` and `undefined` values removed.

```ts
import { compact } from '@rvoh/dream'

compact(['hello', null, 'world'])
// ['hello', 'world']

compact({ hello: 'world', goodbye: null })
// { hello: 'world' }
```
