# string helpers

## camelize

```ts
import { camelize } from '@rvoh/dream'

camelize('hello_world')
// helloWorld
```

## hyphenize

```ts
import { hyphenize } from '@rvoh/dream'

pascalize('helloWorld')
// hello-world
```

## pascalize

```ts
import { pascalize } from '@rvoh/dream'

pascalize('hello_world')
// HelloWorld
```

## snakeify

```ts
import { snakeify } from '@rvoh/dream'

snakeify('helloWorld')
// hello_world
```
