# hyphenize

`hyphenize` converts strings to hyphen-case (kebab-case).

```ts
import { hyphenize } from '@rvoh/dream'

hyphenize('HelloWorld') // 'hello-world'
hyphenize('dream_and_psychic') // 'dream-and-psychic'
```
