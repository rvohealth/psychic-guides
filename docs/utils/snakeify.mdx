# snakeify

`snakeify` converts strings to snake_case.

## Usage

```ts
import { snakeify } from '@rvoh/dream'

snakeify('HelloWorld') // 'hello_world'
snakeify('dream-and-psychic') // 'dream_and_psychic'
```
