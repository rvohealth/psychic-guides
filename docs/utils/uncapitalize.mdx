# uncapitalize

`uncapitalize` converts the first character of a string to lowercase.

```ts
import { uncapitalize } from '@rvoh/dream'

uncapitalize('HelloWorld') // 'helloWorld'
uncapitalize('DreamAndPsychic') // 'dreamAndPsychic'
```
