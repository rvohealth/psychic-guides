# sort

Returns a copy of the array, sorted in ascending order.

```ts
import { sort } from '@rvoh/dream'

sort([2, 1, 3])
// [1, 2, 3]

sort(['world', 'Hello', 'hello', 'World'])
// ['hello', 'Hello', 'world', 'World']
```
