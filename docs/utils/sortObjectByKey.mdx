# sortObjectByKey

Returns a copy of the object with its keys sorted in ascending order.

```ts
import { sortObjectByKey } from '@rvoh/dream/utils'

sortObjectByKey({ c: 3, a: 1, b: 2 })
// { a: 1, b: 2, c: 3 }
```
