# isEmpty

Returns true if the array, object, Set, or Map is empty, false otherwise.

```ts
import { isEmpty } from '@rvoh/dream'

isEmpty([])
// true

isEmpty([1])
// false

isEmpty({})
// true

isEmpty({ hello: 'world' })
// false
```
