# Psychic 2.0 Documentation (Express-based)

:::warning Legacy Version - Psychic 2.0
This documentation is for **Psychic 2.0**, which uses Express as its underlying HTTP framework.

**If you're using Psychic 3.0 or later**, please refer to the [main documentation](/docs/intro) which covers the Koa-based implementation.
:::

## About Psychic 2.0

Psychic 2.0 is built on [Express](https://expressjs.com/), a mature and widely-used Node.js web framework. This version provides:

- Express-based middleware (`(req, res, next)` signature)
- Express request/response objects
- Integration with the Express ecosystem
- Passport.js integration using `express-session`

## Upgrading to Psychic 3.0

If you're considering upgrading to Psychic 3.0, which uses Koa, please see the [migration guide in the CHANGELOG](https://github.com/rvohealth/psychic/blob/main/CHANGELOG.md#300).

Key changes in 3.0 include:
- Express → Koa migration
- Middleware signature changes: `(req, res, next)` → `(ctx, next)`
- Response methods: `res.json()` → `ctx.body = ...`
- HTTP headers are lowercased in Koa
- Passport integration uses `koa-passport` and `koa-session`
