# generating serializers

If you are looking to generate a serializer, you can use the `g:serializer` command provided by dream, like so:

```bash
yarn psy g:serializer User email:string id:string
```

:::tip
For more comprehensive documentation on generating serializers, see our [generating serializer docs](/docs/serializers/generating).
:::
