# new model

A new model can be generated using the CLI

```bash
yarn psy g:model todo user:belongs_to content:string
```

Running this generates a new todo model for us, automatically adding on all of the columns we specified in our generator command, including our belongs_to association to the user model.

```ts
import { DateTime } from 'luxon'
import { dream, Column, BelongsTo, DreamColumn } from 'dream'
import User from './User'

export default class Todo extends Dream {
  public readonly get table() {
    return 'todos' as const
  }

  public id: DreamColumn<Todo, 'id'>
  public content: DreamColumn<Todo, 'content'>
  public createdAt: DreamColumn<Todo, 'createdAt'>
  public updatedAt: DreamColumn<Todo, 'updatedAt'>

  @BelongsTo(() => User)
  public user: User
  public userId: DreamColumn<Todo, 'userId'>
}
```
