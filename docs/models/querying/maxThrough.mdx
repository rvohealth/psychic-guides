# maxThrough

The `maxThrough` method will pluck a field with the highest value from the provided association chain.

```ts
const id = await User.maxThrough('posts', 'posts.id')
// 999
```

Similar to `preload`, `join`, and others, associations can be chained together:

```ts
const id = await User.maxThrough('posts', 'comments', 'comments.id')
// 12
```

Additionally, where statements can be passed to further scope the associations:

```ts
const id = await User.maxThrough('posts', 'comments', { body: null }, 'comments.id')
// 3
```
