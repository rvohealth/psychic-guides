# minThrough

The `minThrough` method will pluck a field with the lowest value from the provided association chain.

```ts
const id = await User.minThrough('posts', 'posts.id')
// 1
```

Similar to `preload`, `join`, and others, associations can be chained together:

```ts
const id = await User.minThrough('posts', 'comments', 'comments.id')
// 3
```

Additionally, where statements can be passed to further scope the associations:

```ts
const id = await User.minThrough('posts', 'comments', { body: null }, 'comments.id')
// 5
```
