# exists

The `exists` method will return true if at least one record is found matching the query. Otherwise, it will return false.

```ts
const exists = await User.exists()
```

## Chaining

Similar to other execution methods, `exists` can be chained together with other query building methods prior to execution.

```ts
const exists = await User.innerJoin('pets', { name: 'I liked it' }).exists()
```
