## use

In addition to the [hooks](/docs/config/hooks) provided by Psychic, you can tap into Koa middleware directly by leveraging the `use` method. This method acts similarly to the Koa `use` API for most use cases:

```ts
psy.use(async (ctx, next) => {
  ctx.body = { hello: 'world' }
})
```

## Injection points

If you need to inject at a specific point during the build cycle, you can provide an optional first argument, which will cause your middleware to run at the specified point:


```ts
// runs before any psychic middleware (i.e. cors) has been processed
psy.use('before-middleware', async (ctx, next) => {
  ctx.body = { hello: 'world' }
})

// runs after all psychic middleware has been provided
psy.use('after-middleware', async (ctx, next) => {
  ctx.body = { hello: 'world' }
})

// runs after all routes in the conf/routes.ts file have been processed
psy.use('after-routes', async (ctx, next) => {
  ctx.body = { hello: 'world' }
})
```
